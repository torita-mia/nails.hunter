# Nails Hunter — Turnera Online

Turnera online para reserva de turnos de nail art.
Belgrano, CABA · La Pampa 2036

## Desarrollo local

```bash
npm install
npm run dev
```

## Deploy en Vercel

1. Subí este proyecto a un repositorio de GitHub.
2. En vercel.com → "Add New Project" → importá el repo.
3. Vercel detecta Vite automáticamente. Click en "Deploy".

¡Listo! No hace falta configurar nada más.
